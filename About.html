Introduction A Logic gate is a kind of the basic building block of a digital circuit having two inputs and one output. The input and output relationship is based on a certain logic. These gates are implemented using electronic switches such as diodes, transistors. But, in practice, the basic logic gates are built using CMOS technology, MOSFET(Metal Oxide Semiconductor FET), FETS. Logic gates are used in microcontrollers, microprocessors, electronic and electrical project circuits, and embedded system applications. The basic logic gates are categorized into seven types as AND, OR, XOR, NAND, NOR, XNOR, and NOT. These are the important digital devices, mainly based on the Boolean function. Logic gates are used to carry out the logical operations on single or multiple binary inputs and result in one binary output. In simple words, logic gates are the electronic circuits in a digital system.
